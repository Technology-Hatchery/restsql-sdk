SQL Resource DELETE Path XML

Method DELETE
Path /res/{resName}/{resId}
Query Params None
Request body One or more child rows to delete of parent matching path params. Request.xsd Content-Type: application/xml
Response body Number of rows deleted. Response.xsd Content-Type: application/xml

Description

Deletes children of a one-to-many hierarchial SQL Resource or associations to children of a many-to-many hierarchical SQL Resource.

Examples

A one-to-many hierarchical SQL Resource (child rows are deleted):
DELETE /restsql/res/LanguageFilm/3 HTTP/1.1
Content-Type: application/xml

<request>
   <film film_id="3" />
   <film film_id="43" />
   <film film_id="67" />
</request>

HTTP/1.1 200 OK
Content-Type: application/xml

<writeResponse rowsAffected="3" />
A many-to-many hierarchical SQL Resource (child rows are disassociated):
DELETE /restsql/res/ActorFilm/123 HTTP/1.1
Content-Type: application/xml

<request>
   <film film_id="3" />
   <film film_id="43" />
   <film film_id="233" />
</request>

HTTP/1.1 200 OK
Content-Type: application/xml

<writeResponse rowsAffected="3" />
