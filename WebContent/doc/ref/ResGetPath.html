SQL Resource GET Path

Method GET
Path /res/{resName}/{resId}
Query Params None
Request body None
Response body Results as XML Response.xsd Content-Type: application/xml

Description

Returns row matching path parameters. Separate compound primary key values by slashes. Empty readResponse document is returned if no rows match criteria.

Examples

GET /restsql/res/Actor/123 HTTP/1.1
HTTP/1.1 200 OK
Content-Type: application/xml

<readResponse>
   <actor actor_id="123" first_name="JULIANNE" last_name="DENCH" />
</readResponse>
a hierarchical SQL resource:
GET /restsql/res/ActorFilm/123 HTTP/1.1
HTTP/1.1 200 OK
Content-Type: application/xml

<readResponse>
   <actor actor_id="123" first_name="JULIANNE" last_name="DENCH">
      <film year="2006" title="ADAPTATION HOLES" film_id="3" />
      <film year="2006" title="ATLANTIS CAUSE" film_id="43" />
      <film year="2006" title="BERETS AGENT" film_id="67" />
      <film year="2006" title="BULL SHAWSHANK" film_id="105" />
      <film year="2006" title="CHOCOLATE DUCK" film_id="148" />
   </actor>
</readResponse>
