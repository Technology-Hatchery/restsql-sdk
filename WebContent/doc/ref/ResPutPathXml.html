SQL Resource PUT Path XML

Method PUT
Path /res/{resName}/{resId}
Query Params None
Request body One or more child rows to update for parent identified in path Request.xsd Content-Type: application/xml
Response body Number of rows updated Response.xsd Content-Type: application/xml

Description

Updates one or more children rows for parent matching path parameters.

Examples

PUT /restsql/res/ActorFilm/123 HTTP/1.1
Content-Type: application/xml

<request>
   <film film_id="3" year="2010" title="ADAPTATION HOLES 2" />
   <film film_id="43" year="2007" />
   <film film_id="67" title="BERETS AGENTS" />
</request>

HTTP/1.1 200 OK
Content-Type: application/xml

<writeResponse rowsAffected="3" />
