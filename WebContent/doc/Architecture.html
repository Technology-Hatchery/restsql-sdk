Requirements

  • Platform Independent - Pure Java solution
  • Java Runtime Environment (JRE) - Tested with 1.6
  • SQL Database - Currently tested with MySQL 5.5, others are planned e.g. PostgreSQL, SQL Server, Oracle
  • JEE Container for Web Service deployment mode - Curently tested with Apache Tomcat 6/7, others are planned e.g. WebLogic and JBoss
  • Java Archive (JAR) Tool for deployment - Tested with 1.6
  • Web Browser for SDK - Tested with IE 7, Firefox 4, Chrome 10
